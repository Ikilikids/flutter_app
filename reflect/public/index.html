プライバシーポリシー

とことん反射神経（以下「本アプリ」）は、ユーザーの個人情報の保護を重要な責務と考え、 以下のとおりプライバシーポリシーを定めます。

1. 取得する情報

本アプリでは、以下の情報を取得する場合があります。

  • アプリの利用状況（操作履歴、クラッシュ情報 等）
  • 端末情報（OS、端末の種類、アプリバージョン 等）
  • アプリ内で生成されるデータ（スコア、設定情報 等）

※ 本アプリでは、氏名・住所・電話番号・メールアドレス等の 個人を直接特定できる情報の入力を求めることはありません。

2. 利用目的

  • アプリ機能の提供および維持
  • 不具合の修正および品質向上
  • 利用状況の分析
  • 不正利用の防止

3. 使用している外部サービス

本アプリでは、以下の外部サービスを利用しています。

Firebase（Google LLC）

  • Firebase Authentication
  • Cloud Firestore
  • Firebase Crashlytics

Firebase は、アプリの動作改善や障害解析のため、 利用状況やクラッシュ情報を収集することがあります。

Google のプライバシーポリシー：
https://policies.google.com/privacy

4. 個人情報の第三者提供

法令に基づく場合を除き、ユーザーの同意なく 取得した情報を第三者に提供することはありません。

5. データの管理

取得した情報は、適切な安全対策を講じた上で管理し、 情報の漏えい・改ざん・不正アクセスの防止に努めます。

6. プライバシーポリシーの変更

本ポリシーは、法令の変更やアプリ内容の変更に応じて、 事前の通知なく改定される場合があります。
